# Revnuvo DNS Intelligence

DNS Intelligence API for agents, MCP clients and automation platforms.

## Endpoints

### Verify Domain

POST /domain/verify

Input:

```json
{
  "domain": "stripe.com"
}
```

### Resolve DNS

POST /domain/dns

Input:

```json
{
  "domain": "stripe.com"
}
```

## Base URL

https://dns.revnuvo.site
